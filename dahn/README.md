# DAHN -- Dynamic, Adaptive, Holon Navigator

This directory contains all of the DAHN code. It is structured in three tiers:

* **_hi_** -- contains the source code for visualizers that present the human interface to the MAP
* **_mid_** -- provides the Rust mid-tier API and implementation for controllers and model objects for DAHN itself
* **_store_** -- contains the Rust / holochain code for the data storage of DAHN elements (e.g., visualizers)
