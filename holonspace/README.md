# holonspace
This microservice supports requests to retrieve and store MAP data. It provides API's to clients (via Conductor) that supports requests and responses containing commands. At this interface, data is represented as self-describing, active holons and holon relationships. Whether or not the data is actually stored in that representation is encapsulated behind this API. Thus, this API can be used as a wrapper on top of existing data stores to make that data available to clients as self-describing, active holons and holon relationships.
